`bool`
>Stores boolean value - *true* or *false*
>
>Range ==  -2<sup>31</sup> : 2<sup>31</sup>-1
>
>Memory == 32 bits

    #include <cs50.h>

    bool isTrue = true;

`string`
>Stores series of characters - words, sentences, paragraphs, etc.

    #include <cs50.h>

    string name = "Alice";
